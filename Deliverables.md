## ** Deliverables**

Participants are expected to submit the following deliverables:

### **1. Backend Service**
Participants should provide a backend service that:

- Accepts candidate details, job role, or resume as input.  
- Automatically generates and asks relevant interview questions.  
- Analyzes candidate responses and provides an evaluation or score.  

**Example:**
- Candidate: John Doe  
- Role: Software Developer  
- Question: Explain the concept of REST APIs.  
- Evaluation Score: 8/10  
- Feedback: Good understanding of API concepts.  

---

### **2. Source Code & Documentation**
Participants must submit the complete source code along with documentation explaining:

- System architecture  
- AI models or APIs used  
- Steps to run the project  
- Example inputs and outputs  

---

### **3. Demo Presentation**
Participants should present:

- A working demo of the AI interviewer  
- Explanation of the interview and evaluation approach  
- Possible future improvements or enhancements
